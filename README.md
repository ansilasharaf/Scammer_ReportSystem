# Scammer_ReportSystem



Project Overview

The Scammer Reports System is an AI-based web application developed to create a secure and reliable academic environment by preventing scam-related and misleading content sharing among students.

With the increasing use of online resources for education, students often face risks such as fake websites, misleading study materials, and scam links. Traditional learning platforms do not provide proper verification mechanisms, which may lead to misinformation and unsafe browsing.

This project focuses on developing a smart web platform that integrates Artificial Intelligence to provide secure academic support, verify shared content, and assist students through an intelligent chatbot system.


Objective:

The main objectives of this project are to:

* Improve academic support using AI technology
* Prevent scam and fake content sharing
* Provide reliable, content-based student assistance
* Ensure safe and verified educational resource sharing
* Create a centralized platform for academic interaction


Features Implemented (Web Side):

1️⃣ User Management

* Manage Admin, Staff, and Student accounts
* Secure login and registration system
* Role-based access control

2️⃣ AI Chatbot System

* Chatbot answers student queries
* Uses staff-uploaded study materials
* Provides accurate, syllabus-based response

3️⃣ Study Material Management

* Staff can upload notes and documents
* Students can access learning resources
* Update and manage study materials


4️⃣ Scam Detection & Link Verification

* Students can share educational links
* System verifies links using Machine Learning
* Blocks scam or irrelevant content
* Allows only safe and useful links


5️⃣ Scammer Reporting System

* Students can report suspicious links or content
* Admin reviews reported cases
* Maintains scam report records


6️⃣ Alert & Monitoring System

* Notify admin about suspicious activities
* Monitor system usage and interactions
* Ensure safe academic environmen

Technologies Used:

* Frontend: HTML, CSS, Bootstrap, JavaScript
* Backend: Python (Django)
* Database: MySQL
* AI Integration: Chatbot + Machine Learning Model
* Tools: PyCharm, Android Studio

System Workflow:

* Staff uploads study materials into the system
* Chatbot processes and understands the content
* Student asks questions → chatbot provides answers based on materials
* Student shares a link → system verifies using ML model
* If link is safe → allowed
* If link is suspicious → blocked and reported
* Admin reviews reports and manages system activities


Project Modules:

* Admin Module
* Staff Module
* Student Module


Advantages of the System:

* Prevents scam and fake content
* Provides accurate academic assistance
* Improves student learning experience
* Ensures safe information sharing
* Reduces dependency on external sources

Conclusion:

The Scammer Reports System provides a smart and secure platform for academic interaction by combining AI chatbot technology with scam detection and content verification. It enhances learning quality, ensures digital safety, and creates a reliable educational environment for students.
